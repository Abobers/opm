Что нового в opm 1.2?

1. Немного ускорено чтение команд


2. Новые команды:
2.1. paths
Показывает в виде таблицы пути до библиотек для всех версий Lazurite.

2.2. outdated
Выводит в виде таблицы все библиотеки, доступные для обновления.

2.3. updateall
Обновляет все библиотеки до последней версии.


3. Вместо указания конкретной версии можно использовать слова:
last, latest - последняя версия (2.7.1)
prelast, prelatest - предпоследняя версия (2.7)
this, current - текущая версия


4. команда path <version> <path>
Теперь при смене пути до библиотек все файлы библиотек перемещаются в новую папку автоматически.
