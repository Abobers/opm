Как использовать пакетный менеджер?

1. Скачайте пакетный менеджер opm 1.2

2. Разархивируйте (вместе с config.json) и запустите opm.exe

3. Введите команду "path 2.7.1 path/to/lazurite_libs"
* Можно указать любую другую версию Lazurite 2.0+, а не 2.7.1
* Можно указать другую папку, но она должна заранее существовать

4. Введите команду "lazurite version 2.7.1"
* Можно указать любую другую версию Lazurite 2.0+, но нужно заранее указать путь для библиотек
* Две версии Lazurite не могут иметь одну папку для хранения библиотек

5. Готово! Теперь вы можете скачивать библиотеки:
install math
install Arrays++
Все команды можно увидеть прописав команду "help"



Основные команды:
install <name>
Устанавливает библиотеку.

update <name>
Обновляет библиотеку.

delete <name>
Удаляет библиотеку.

list
Таблица со всеми установленными библиотеками.
